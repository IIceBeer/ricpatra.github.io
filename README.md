# Ric Patra Website

Premium portfolio website for Ric Patra, built with Next.js, React, and a custom orange-white visual system.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run start
```

## Project structure

- `app/`
- `components/`
- `lib/`
- `public/`
- `scripts/`

## Notes

- This GitHub repository stores the website source code.
- The live website is deployed separately.
